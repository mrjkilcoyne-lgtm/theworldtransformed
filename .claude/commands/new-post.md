# Create New Blog Post

Please create a new blog post with the following structure:

1. **Determine the filename**: Ask me for the post title and suggest a kebab-case filename
2. **Create the file** in `src/content/blog/` (or appropriate location)
3. **Include proper frontmatter**:
   - title: [Post title]
   - date: [Today's date in YYYY-MM-DD format]
   - description: [Brief description]
   - tags: [Relevant tags from: economics, policy, technology, philosophy, coordination]
   - draft: false

4. **Use this template**:

```mdx
---
title: "Post Title Here"
date: YYYY-MM-DD
description: "Brief description of the post"
tags: ["economics", "policy"]
draft: false
---

## Introduction

[Opening paragraph]

## Main Content

[Content here]

## Conclusion

[Closing thoughts]

---

*Last updated: {date}*
```

5. **Verify**: Check that the file was created correctly and is in the right location

Ask me for any missing information before proceeding.
